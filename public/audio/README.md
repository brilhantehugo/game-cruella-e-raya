# Arquivos de BGM

Baixe 4 faixas CC0/CC-BY de **opengameart.org** e salve aqui com os nomes abaixo:

| Arquivo | Onde buscar |
|---------|------------|
| `bgm_menu.mp3` | Busca: "chiptune menu loop CC0" |
| `bgm_world1.mp3` | Busca: "chiptune platformer loop CC0" |
| `bgm_boss.mp3` | Busca: "chiptune boss battle CC0" |
| `bgm_fanfare.mp3` | Busca: "chiptune level complete fanfare CC0" |

O jogo funciona sem esses arquivos (Phaser ignora áudio ausente silenciosamente).
