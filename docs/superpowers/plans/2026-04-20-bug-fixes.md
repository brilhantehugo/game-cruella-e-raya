- [ ] **Step 3: Atualizar o texto de instruções (linha 167)**

- [ ] **Step 4: Adicionar handler de ENTER após o handler de ESC**

Localizar o bloco:
```typescript
    // ── Tecla ESC ───────────────────────────────────────────────────────
    this.input.keyboard!.once('keydown-ESC', () => {
      this.scene.start(KEYS.MENU)
    })
```
Substituir por:
```typescript
    // ── Tecla ESC ───────────────────────────────────────────────────────
    this.input.keyboard!.once('keydown-ESC', () => {
      this.scene.start(KEYS.MENU)
    })

    // ── Tecla ENTER — inicia o level atual do perfil ─────────────────────
    this.input.keyboard!.once('keydown-ENTER', () => {
      const profile = profileManager.getActive()
      const levelId = profile?.currentLevel
      if (levelId && profileManager.isUnlocked(levelId)) {
        this._startLevel(levelId)
      }
    })
```

### Task 2: Bug 2 — Remover collider entre enemyGroup e platformLayer

**Files:**
- Modify: `src/scenes/GameScene.ts:658`
- Test: `tests/GameScene.test.ts` (existente)

**Contexto:**
- O método `_setupCollisions()` em `GameScene.ts` fica em torno da linha 650
- O corpo físico dos HumanEnemies (Hugo, Hannah, etc.) tem `halfHeight=42` (body criado a scale=2 com textura de 42px de altura)
- Esse body se sobrepõe verticalmente com as plataformas do row 11 (`y=360–376`) para inimigos perto de `x=1600–1760`
- O collider empurra inimigos para baixo, conflitando com o groundLayer → afundam
- Nenhum inimigo humano jamais spawna em plataformas elevadas, então este collider é desnecessário

- [ ] **Step 1: Localizar a linha exata**

```bash
grep -n "collider.*enemyGroup.*platformLayer\|platformLayer.*enemyGroup" src/scenes/GameScene.ts
```
Esperado: uma linha, algo como `this.physics.add.collider(this.enemyGroup, this.platformLayer)`.

- [ ] **Step 2: Remover a linha**

Em `src/scenes/GameScene.ts`, localizar e remover:
```typescript
    this.physics.add.collider(this.enemyGroup, this.platformLayer)
```

O bloco antes da mudança fica assim:
```typescript
    this.physics.add.collider(this.enemyGroup, this.groundLayer)
    this.physics.add.collider(this.enemyGroup, this.platformLayer)  // ← REMOVER esta linha

    // Decorações sólidas...
```

Depois da mudança:
```typescript
    this.physics.add.collider(this.enemyGroup, this.groundLayer)

    // Decorações sólidas...
```
